UI Developer

We are seeking a talented UI Developer to join our pasta store team. As a UI Developer, you will be responsible for designing and implementing user interfaces that enhance the user experience on our website. Your primary focus will be on creating visually appealing and user-friendly interfaces that align with our brand identity.

Responsibilities:

  • Collaborate with the design team to translate wireframes and mockups into interactive web pages.
  • Develop responsive and accessible user interfaces using HTML, CSS, and JavaScript.
  • Ensure cross-browser and cross-device compatibility of the website.
  • Optimize web pages for maximum speed and scalability.
  • Conduct usability testing and incorporate user feedback into the design process.
  • Stay up-to-date with industry trends and best practices in UI development.

Requirements:

  • Proven experience as a UI Developer or similar role.
  • Proficiency in HTML, CSS, JavaScript, and other relevant web technologies.
  • Strong understanding of responsive design principles and mobile-first development.
  • Experience with front-end frameworks such as Bootstrap.
  • Knowledge of UI/UX design principles and best practices.
  • Excellent problem-solving and communication skills.

If you are passionate about creating engaging user experiences and have the technical skills to bring designs to life, we would love to hear from you. Join our team and contribute to the success of our pasta store!

Node.js Developer

We are looking for a skilled Node.js Developer to join our pasta store team. As a Node.js Developer, you will be responsible for developing server-side logic and building scalable web applications to support our online pasta store.

Responsibilities:

  • Design and develop robust and scalable backend applications using Node.js and related frameworks.
  • Collaborate with cross-functional teams to define and implement APIs for seamless integration with frontend components.
  • Write efficient and reusable code following best practices and coding standards.
  • Optimize application performance and troubleshoot any issues or bugs.
  • Ensure the security and integrity of data by implementing appropriate security measures.
  • Participate in code reviews and provide constructive feedback to team members.
  • Stay up-to-date with the latest trends and advancements in Node.js and backend development.

Requirements:

  • Strong proficiency in Node.js and JavaScript.
  • Experience with backend frameworks such as Express.js.
  • Knowledge of database technologies like MongoDB or MySQL.
  • Understanding of RESTful API design principles.
  • Familiarity with version control systems such as Git.
  • Experience working in an Agile/Scrum development environment.
  • Good problem-solving and communication skills.

If you are passionate about building scalable and efficient web applications using Node.js, and have a keen interest in the food industry, we would love to have you on our team. Join us and contribute to the success of our pasta store!
